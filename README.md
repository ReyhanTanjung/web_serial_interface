# Installation Guide
## Backend Javascript
```
npm install
```
## Backend Python
```
cd rtsp_streamer
python -m venv env
env\Scripts\activate
pip install -r requirements.txt
```

# Usage
## 1. Open first terminal
```
node .\src\index.js
```
## 2. Open second terminal
```
cd rtsp_streamer
env\Scripts\activate
python rtsp_viewer.py
```
